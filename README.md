# 🏎️ 马里奥卡丁车 (Mario Kart Racing)

一款基于 HTML5 Canvas 开发的俯视角赛车游戏，灵感来自经典的马里奥赛车系列。

![Game Preview](game-preview.png)

## 🎮 游戏特色

### 核心玩法
- **俯视角赛车**：经典的俯视视角，清晰掌握全局赛道情况
- **流畅操控**：支持键盘精准控制，体验真实的赛车物理
- **漂移系统**：按住 Shift 键进行漂移，过弯更爽快

### 赛道设计
- 精心设计的闭合环形赛道，包含多个弯道
- 草地、泥土等不同地形影响赛车速度
- 障碍物（树木、岩石）增加挑战性
- 起点线采用经典黑白格设计

### 角色与车辆
- **Mario** (🔴 红色赛车) - 玩家操控
- **Luigi** (🟢 绿色赛车) - AI对手
- **Peach** (🩷 粉色赛车) - AI对手
- **Bowser** (🟤 棕色赛车) - AI对手

### 道具系统
游戏包含5种经典道具：

| 道具 | 效果 |
|------|------|
| 🍄 加速蘑菇 | 瞬间加速，获得爆发性速度 |
| 🍌 香蕉皮 | 在身后放置陷阱，让对手打滑 |
| 🐢 龟壳 | 向前发射攻击性投射物 |
| ⭐ 无敌星 | 短时间内无敌并加速 |
| ⚡ 闪电 | 缩小所有对手 |

### 游戏元素
- **计圈系统**：3圈定胜负，实时显示当前圈数
- **排名系统**：动态计算所有赛车位置
- **AI对手**：3个智能AI对手，具有不同的驾驶风格
- **碰撞检测**：赛车之间、障碍物、道具的全面碰撞

### 视觉效果
- 🎨 色彩鲜艳的卡通风格
- ✨ 漂移火花粒子效果
- 💨 加速尾焰特效
- 🎁 道具箱旋转动画
- 📊 实时UI显示（速度表、圈数、排名）

## 🎯 操作指南

### 基础控制
| 按键 | 功能 |
|------|------|
| `W` 或 `↑` | 加速 |
| `S` 或 `↓` | 刹车/倒车 |
| `A` 或 `←` | 左转 |
| `D` 或 `→` | 右转 |
| `Shift` | 漂移（过弯时按住） |
| `空格键` | 使用道具 |

### 游戏技巧
1. **漂移过弯**：接近弯道时按住 Shift + 方向键，可以更快通过弯道
2. **道具策略**：在直道上使用加速蘑菇，在弯道前放置香蕉皮
3. **抄近道**：注意赛道边缘的草地会减速，尽量保持在赛道上
4. **防御驾驶**：注意身后的龟壳，可以通过急转弯躲避

## 🚀 快速开始

### 方法一：直接打开
1. 下载项目文件夹
2. 双击 `index.html` 文件
3. 在浏览器中开始游戏

### 方法二：本地服务器
```bash
cd mario-kart-racing
python -m http.server 8000
# 然后访问 http://localhost:8000
```

## 🛠️ 技术栈

- **纯 HTML5 + Canvas**：不依赖任何外部游戏引擎
- **原生 JavaScript**：ES6+ 语法，面向对象设计
- **响应式设计**：自适应不同屏幕尺寸
- **单文件结构**：简洁的模块化代码组织

### 代码结构
```
mario-kart-racing/
├── index.html      # 游戏主页面和UI
├── game.js         # 游戏核心逻辑
└── README.md       # 项目说明
```

## 🎨 自定义修改

### 调整难度
在 `game.js` 中修改以下配置：
```javascript
const GAME_CONFIG = {
    TOTAL_LAPS: 3,        // 比赛圈数
    PLAYER_COUNT: 4,      // 参赛人数
    TRACK_WIDTH: 120,     // 赛道宽度
};
```

### 修改赛车属性
```javascript
this.maxSpeed = 12;       // 最大速度
this.acceleration = 0.3;  // 加速度
this.turnSpeed = 0.08;    // 转向速度
```

### 添加新道具
在 `ITEM_TYPES` 中添加：
```javascript
NEW_ITEM: { 
    emoji: '🆕', 
    name: '新道具', 
    effect: 'custom_effect' 
}
```
